# Evan 2.0 - AI Chatbot

Evan 2.0 is a personal chatbot project designed to mimic the personality of a chill young teenager texting a friend. 
It was **slightly assisted by AI** during development to refine prompts and response patterns, but the core personality and logic were crafted manually.

## Usage

Clone this repository and run the bot locally:

```bash
python evan_bot.py
