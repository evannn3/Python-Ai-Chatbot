# Evan 2.0 - AI Chatbot

Evan 2.0 is a personal chatbot project designed to mimic the personality of a chill young teenager texting a friend. 
It was **slightly assisted by AI** during development to refine prompts and response patterns, but the core personality and logic were crafted manually.
Version no.1 of the bot (Better versions to come in the future).
## Bot model 

Model --> https://huggingface.co/TinyLlama/TinyLlama-1.1B-Chat-v1.0

## Features

- 'Tell me a joke function' to keep users entertained and add humanistic realism.
- Memory system to remember basic facts about the user.
- Mimics a young teenager’s texting style, staying relaxed and friendly.
